## Getting Started

Follow these steps to set up the project locally.

### Prerequisites

- Node.js (>=16.x)
- npm or yarn

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/gdg-klef/Dashboard_mock.git
   cd Dashboard_mock

2. Install dependencies:
    ```bash
    npm install
    ```
   or

   ```bash
   yarn install
   ```


4. Start the deployment server
   ```bash
   npm run dev
   ```
   or

   ```bash
   yarn dev
   ```
